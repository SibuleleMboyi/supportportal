To see the contents of the generated JWT :-

Run the Application.
Launch "Thunder Client/ Postman" and send login Request.
In the HTTP response window, go to the "Headers".
Copy the "jwt-token".


Go to JWT.io website.
Scroll to the JWT posted in the Site and replace it with the Copied "jwt-token".


TO VERIFY THE TOKEN :-

Go to the "applications.yml" and copy the Secrete.
Paste it in the secrete field in JWT.io website.
